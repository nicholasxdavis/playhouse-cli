## Problem

The paste dependency at version 1.0.15 has a known security vulnerability (RUSTSEC-2024-0436).

## Impact

Potential security risk or compiler-level exploits in dependent macro evaluations.

## Suggested fix

1. Check Cargo.toml and Cargo.lock to see where paste is required.
2. Upgrade paste to a patched version (e.g. 1.0.16 or later) to resolve the vulnerability.

**Files:** Cargo.toml, Cargo.lock

## Acceptance criteria

- [ ] paste dependency is upgraded and dependency audits no longer report RUSTSEC-2024-0436

## Priority

P1 - should ship (dependency vulnerability)
