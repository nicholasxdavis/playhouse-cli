## Problem

The lru package at version 0.12.5 has a known security vulnerability (RUSTSEC-2026-0002 / GHSA-rhfx-m35p-ff5j).

## Impact

Potential security risk or instability if vulnerable logic in lru is triggered by the application.

## Suggested fix

1. Check Cargo.toml and Cargo.lock to see where lru is required.
2. Upgrade the dependency to a patched version (e.g. 0.12.6 or later) where the vulnerability is resolved.

**Files:** Cargo.toml, Cargo.lock

## Acceptance criteria

- [ ] lru dependency is updated and cargo audit / osv-scanner no longer reports RUSTSEC-2026-0002
